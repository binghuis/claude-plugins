# git

git 分支日常操作 skill 集合。

## Skills

- **[branch](skills/branch/SKILL.md)**：解析用户的分支意图，路由到 `switch`（命中已有分支）或 `new-branch`（新建）
- **[switch](skills/switch/SKILL.md)**：切到已存在的分支并保留 WIP（auto-stash，pop 前询问）
- **[new-branch](skills/new-branch/SKILL.md)**：从当前 HEAD 新建分支并切过去，WIP 随之带过
- **[release-today](skills/release-today/SKILL.md)**：切到/新建今日日抛分支 `release/MM-DD`
